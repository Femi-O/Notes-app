How to animate a modal with CSS

  1. Make your div to display with ngIf.
  2. Make a backdrop div to emulate the shadow that comes under a modal, tied to the same ngIf.
  3. A function to toggle the display of the modal and its backdrop between "true" and "false".
  4. Call the toggle function on click of the backdrop and anywhere else that dismisses the modal.
  5. For the animation of the modal add the following css to the modal: transition: transform 0.3s ease-in-out;   transform: translateY(100%);
  6. Add the following css to the modal's open state: transform: translateY(0);
  7. Ensure the HTML tag is properly given the necessary css class and ngClass (to control state of class when opened and closed)
